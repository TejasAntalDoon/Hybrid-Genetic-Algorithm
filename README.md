# Hybrid Genetic Algorithm

## Overview

This project implements a Hybrid Genetic Algorithm (HyGA) integrated with Deep Learning techniques to optimize model performance and improve prediction accuracy. The project demonstrates how evolutionary algorithms can be combined with machine learning models to search for optimal solutions efficiently.

## Features

* Hybrid Genetic Algorithm implementation
* Deep Learning model integration
* Hyperparameter optimization
* Performance evaluation and analysis
* Visualization of results

## Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* TensorFlow / Keras

## Installation

Clone the repository:

```bash
git clone https://github.com/TejasAntalDoon/Hybrid-Genetic-Algorithm.git
cd Hybrid-Genetic-Algorithm
```

Install the required libraries:

```bash
pip install numpy pandas matplotlib scikit-learn tensorflow
```

## Usage

1. Open Jupyter Notebook.
2. Open `HyGA_Proj.ipynb`.
3. Run all cells sequentially.
4. Review the generated results and visualizations.
